อ่านกติกาและทำความเข้าใจเพื่อผลประโยชน์ของท่าน

  1. รายการการเล่น Internet Betting ถือเป็นผลสำเร็จเมื่อได้รับการยืนยันว่า “ สำเร็จ ” โดยรายการที่เล่นมาไม่สามารถยกเลิกรายการเล่นได้ และสามารถตรวจสอบการแทงได้อีกครั้ง ในเมนู “ รายการเล่น ” และโปรดตรวจสอบรายการแทงของท่านเพื่อความถูกต้องทุกครั้ง
  2. หากเกิดปัญหาเกี่ยวกับ Internet Betting กรุณาโทรแจ้งทางออฟฟิตก่อนผลรางวัลออก 30 นาที มิฉะนั้น ทางบริษัทจะไม่รับผิดชอบ ไม่ว่าในกรณีใดๆทั้งสิ้น
  3. เมื่อมีการเล่น Internet Betting ให้ทำการ save หรือ print รายการเล่นเก็บไว้เป็นหลักฐานเพื่อตรวจสอบความถูกต้อง หากเกิดความผิดพลาดแล้วทางลูกค้าไม่มีหลักฐานเพื่อยืนยัน ความถูกต้อง ทางบริษัทขอสงวนสิทธิ์ใช้หลักฐานที่ทางบริษัทมีอยู่ในการยืนยันความถูกต้อง
  4. หากมีการเปลี่ยนแปลงใดๆ ก็ตาม ในกรณีต่างๆ ทางบริษัทจะแจ้งให้ทราบผ่านทาง website หรือทางบริษัทจะโทรแจ้งผ่านทาง Agent ตัวแทนของท่าน
  5. สำหรับการแทงหวยรัฐบาล จะใช้
    - ผลรางวัลที่ 1 ตัวเลข 3 ตัวท้ายของรางวัลที่ 1 เป็นผลของ 3 ตัวบน
    - ผลรางวัลที่ 1 ตัวเลข 2 ตัวท้ายของรางวัลที่ 1 เป็นผลของ 2 ตัวบน
    - ผลรางวัล 2 ตัวล่าง จะใช้รางวัลเลขท้าย 2 ตัว
    - ผลรางวัล 3 ตัวหน้าบน จะเป็นรางวัลของ 3 ตัว หน้า - หลัง
  6. บริษัทขอสงวนสิทธิ์ที่จะยกเลิกสมาชิกที่ค้างชำระและเพื่อป้องกันการกลับมาเล่น
  7. บริษัทจะไม่รับผิดชอบการขโมย Username และ Password ของสมาชิก สมาชิกต้องระมัดระวังและรับผิดชอบต่อข้อมูลส่วนตัวของท่าน โดยต้องหมั่นเปลี่ยน password บ่อยๆ เพื่อความปลอดภัย
  8. Agent ต้องดูแลรับผิดชอบต่อสมาชิกของท่านด้วยความซื่อตรงต่อข้อมูลทั้งหมดของสมาชิก
